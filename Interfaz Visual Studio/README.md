
# Proyecto Final - Procesamiento Digital de Imágenes

## Interfaz gráfica para el procesamiento de imágenes médicas DICOM
Este repositorio contiene el desarrollo de la interfaz gráfica del Proyecto Final de la materia de Procesamiento Digital de Imágenes.

La aplicación fue desarrollada en Python utilizando CustomTkinter y permite visualizar estudios médicos en formato DICOM, navegar entre sus diferentes cortes y aplicar diversas técnicas clásicas de procesamiento digital de imágenes de manera interactiva.

## Funciones implementadas

- Visualización de estudios DICOM
- Navegación entre cortes
- Visualización de histogramas
- Ecualización del histograma
- Filtro de media
- Filtro Gaussiano
- Resta Original - Gaussiano
- Operador Sobel
- Operador Laplaciano
- Segmentación por umbral
- Operaciones lógicas AND y OR
- Transformada de Fourier
- Inversión de imagen
- Ajuste interactivo del umbral
- Guardado de imágenes procesadas

## Tecnologías utilizadas

- Python
- OpenCV
- NumPy
- Matplotlib
- CustomTkinter
- Pillow
- pydicom
